# Car-Insurance-Tableau
## 📊 Car Insurance Claims Analysis

This project explores the **Car Insurance Claims dataset** to uncover meaningful insights and trends.

### 🔍 Key Highlights
- Conducted in-depth analysis of claims data to identify patterns and anomalies  
- Built **interactive and responsive dashboards** for better data visualization  
- Presented a **logical storytelling flow** to communicate findings effectively  

### 📈 Insights & Outcomes
- Identified key trends in insurance claims behavior  
- Highlighted critical factors impacting claim frequency and outcomes
